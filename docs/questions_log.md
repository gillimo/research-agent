Questions Log
=============

Purpose
- Track blockers and open questions.
- Close resolved items daily to keep the backlog clean.

How To Log A Question
- Use `scripts/log_question.py --text "your question"` to add a new entry.
- Use `scripts/log_question.py --close <id>` to close an entry when resolved.

Entries are stored in `logs/questions.ndjson`.
